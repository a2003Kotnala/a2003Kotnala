## Tech Stack

### 1. Machine Learning
Supervised and unsupervised learning, reinforcement learning fundamentals, RLHF, PCA, t-SNE, UMAP, time series forecasting, hyperparameter optimization using Grid Search and Random Search, ensemble techniques, and evaluation metrics.

### 2. Deep Learning
ANN, CNN, ResNet, optimizers like Adam and AdaGrad, backpropagation, RNN, LSTM, GRU, Seq2Seq models, transformers, Hugging Face, and self-attention.

### 3. Computer Vision / OCR
Computer vision fundamentals, OCR, YOLO, and Tesseract.

### 4. NLP
Tokenization, embeddings, vector databases like ChromaDB and Pinecone, language models, RAG, RAGAS, and embedding models.

### 5. GenAI
LangChain, LangGraph, LlamaIndex, Ollama, and speech processing using ffmpeg and Whisper.

### 6. MCP / MLOps / Backend
Basic understanding of MCP and MLOps, Docker, PyTorch, FastAPI, Pydantic, Alembic, REST APIs, Uvicorn, Ngrok, PostgreSQL, AWS SageMaker, and EC2.
